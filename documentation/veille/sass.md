# Utilisation de SASS

## Objectif

Nous avons choisi d’utiliser SASS
afin d’améliorer l’organisation
et la maintenance du CSS.

SASS permet de structurer plus facilement
les fichiers de style de notre boutique en ligne.

## Avantages de SASS

SASS permet :

- d’utiliser des variables
- de séparer le CSS en plusieurs fichiers
- de limiter les répétitions
- de rendre le code plus lisible
- de faciliter la maintenance du projet

## Couleurs principales du projet

Nous utiliserons des variables SASS
pour gérer la charte graphique du site.

Exemples :

- bleu principal : #00439C
- bleu hover : #005BEA
- orange accent : #FF4500
- fond clair : #F4F6F8
- texte principal : #1A1D20

## Organisation du CSS

Le projet sera séparé en plusieurs fichiers SASS :

- variables
- header
- footer
- dashboard
- responsive
- boutons

## Responsive et Mobile First

SASS sera également utilisé
pour organiser les media queries
du responsive mobile first.
