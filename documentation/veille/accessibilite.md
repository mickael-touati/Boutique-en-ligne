# Accessibilité du dashboard administrateur

## Objectif

Nous voulons rendre le dashboard administrateur
accessible et simple à utiliser pour tous les utilisateurs.

## Les choix réalisés

Nous utiliserons :

- des couleurs lisibles avec un bon contraste
- des boutons visibles
- des textes assez grands
- des labels sur les formulaires
- des balises HTML sémantiques
- des attributs alt pour les images

## Navigation clavier

Le dashboard devra être utilisable au clavier
afin d’améliorer l’accessibilité.

## Structure HTML

Nous utiliserons des balises adaptées :

- header
- nav
- main
- section
- footer

## Responsive

Le dashboard sera pensé en mobile first
afin d’être utilisable sur téléphone et tablette.
