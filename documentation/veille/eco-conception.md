# Éco-conception de l'application

## Objectif

Nous avons essayé de concevoir une boutique en ligne
plus légère et optimisée afin de limiter
la consommation de ressources.

Nous avons également cherché à améliorer
les performances et l’accessibilité du site.

## Mobile First

Le projet est développé en mobile first.

Nous avons commencé par concevoir
l’affichage mobile avant les versions tablette
et desktop.

Les media queries seront principalement utilisées
pour les grands écrans.

## Optimisation des images

Les images utilisées sur le site
seront compressées et redimensionnées
afin de limiter leur poids.

Cela permet d’améliorer le temps de chargement
des pages.

## Utilisation de SASS

Le CSS du projet sera développé avec SASS.

Cela permet :

- une meilleure organisation
- une maintenance plus simple
- moins de répétitions dans le code

## Optimisation JavaScript

Nous éviterons d’utiliser trop de scripts inutiles
afin de limiter le poids du site.

Le JavaScript sera principalement utilisé
pour les fonctionnalités asynchrones importantes.

## Accessibilité et performance

Nous avons choisi des couleurs lisibles
et une structure claire afin d’améliorer
le confort de navigation des utilisateurs.
