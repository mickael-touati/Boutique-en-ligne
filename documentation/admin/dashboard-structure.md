# Structure du Dashboard Administrateur

## Objectif

Le dashboard administrateur permettra
de gérer les produits, catégories et stocks
de la boutique en ligne.

L’administrateur pourra ajouter,
modifier et supprimer des données
grâce à une interface simple.

## Pages principales du dashboard

Le dashboard sera composé de plusieurs pages :

- dashboard principal
- gestion des produits
- gestion des catégories
- gestion des stocks
- gestion des commandes
- connexion administrateur

## Structure visuelle

Le dashboard utilisera :

- une sidebar de navigation
- une zone principale de contenu
- des tableaux pour afficher les données
- des formulaires pour le CRUD

## Menu de navigation

La sidebar contiendra :

- Dashboard
- Produits
- Catégories
- Stocks
- Commandes
- Déconnexion

## Accessibilité

Le dashboard sera développé
avec une structure HTML sémantique.

Les boutons et formulaires
seront accessibles au clavier
et lisibles pour tous les utilisateurs.

## Responsive

Le dashboard sera pensé en mobile first
afin d’être utilisable sur mobile,
tablette et ordinateur.
